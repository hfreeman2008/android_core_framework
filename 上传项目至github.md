上传项目至github


#验证成功的方法
假设已经有github账号

# 1.github新建项目
点击Repositories--New


输入Repository name，点击Create repossitory.


# 2.GitHub Desktop
点击任意项目的code--Open with GitHub Desktop，下载安装GitHub Desktop，同步github账号。


# 3.GitHub Desktop提交同步代码
## add commit代码:
输入Description，然后Commit to main.

## push代码：
Repository--push
